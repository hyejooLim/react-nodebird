# 🐦 react-nodebird

react-nodebird는 `React`, `Next.js` 프레임워크를 사용하여 만든 SNS 서비스입니다. </br></br>
(제로초 님의 강의를 통해 학습하며 개발한 클론코딩 프로젝트입니다.)

</br></br>

## 🌼 기술 스택

</br>

> ### Frontend
* React Hooks
* Redux
* Redux saga
* Next.js(SSR)
* Ant Design
* Styled Components
* Immer

</br>

> ### Backend
* Node.js
* Express
* Sequelize(MySQL)
* SWR

</br></br>
